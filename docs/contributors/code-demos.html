    Creating a Demo (annoted sketch)

    Anyone can create and share "sketches" (single file web sites) in netnet by writing some code in netnet's editor and then saving it (either as a sharable URL or downloadable html file). These sketches can also become "demos", with annotations explaining how different parts of the code works. These annotated sketches, or "demos", can be shared as URLs like any other netnet sketch, but you can also contribute them to netnet.studio for inclusino in the "Code Demos" widget.
