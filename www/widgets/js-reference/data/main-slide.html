Unlike HTML and CSS, JavaScript is a programming language. There are loads of different programming languages out there (Python, Java, C++, Rust, C, etc) and they all basically do the same thing: allow you to write programs that get the computer to do what you want it to do. All programming languages contain the same core concepts (variables, functions, loops, conditional statements, etc) but they all have slightly different ways of expressing these concepts, which is to say, they all have different syntax and APIs.

Like any other programming language, JavaScript can be used to create all sorts of things outside of the web browser, from native apps to robots, you can even use it to script other programs like the Adobe Creative Suite and Microsoft Office. All that said, JavaScript was originally invented for the Web and it's inside the browser where its creative possabilities really shine!
