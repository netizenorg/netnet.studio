"AI"

Notes Concerning Artificial "Intelligence"

Today, algorithms aren’t only written, they’re “trained” through a process called “machine learning”. These new “AI” systems, like many technologies before them (smartphones, the Internet, computers), will undoubtably reshape creative practices, though exactly how and to what extent is still anyone's guess. We’ve assembled some interactive notes to help you incorporate these tools into your practice in a way that supports both your learning and making, rather than undermining it.

Hypertext Markup Language

As a creative medium, the Web has three core coding languages at the center of its craft: HTML, CSS and JavaScript. The most fundamental of these is HTML, or Hypertext Markup Language. HTML is used to create the structure of a Web project.

CSS



Cascading Style Sheets

Cascading Style Sheets, or CSS, was a language introduced to the Web in the mid 90s for the purpose of separating a webpage's "form" or "presentation" from it's "structure" and "content". Initially it handled things like layout, color and fonts, but today, it has evolved to become a much more expressive language capable of creating interactive effects and animations.

JavasScript is the Web's defacto programming language, although you can use it in other contexts (robotics, AI, games, etc) it was first invented to add interactive and dynamic capabilities to the Web, converting our hypertext "documents" into hypermedia "applications".
